# 📥 Download Dataset

This project uses historical stock market datasets that are hosted separately on **Kaggle** because the complete datasets exceed GitHub's file size limitations.

---

# 📊 Dataset Information

The dataset includes historical stock market data used throughout this project, including:

- 🇺🇸 US500 Stocks
- 🇮🇳 NIFTY50 Stocks
- 🌍 7000Stocks Dataset

These datasets are used for:

- Data preprocessing
- Feature engineering
- Model training
- Performance evaluation
- Web application prediction

# 🔗 Download Dataset

Download the complete dataset from Kaggle:

**Kaggle Dataset**

# 📁 After Downloading

Extract the downloaded ZIP file.

Move the extracted files into the appropriate project directory.

Example:

```text
advanced_models/
│
├── data/
│   ├── 7000Stocks_all.csv
│   ├── NIFTY50_all.csv
│   ├── US500_all.csv
│   └── ...
```

# 📌 Why Isn't the Dataset Included?

The complete datasets are not stored in this GitHub repository because they exceed GitHub's recommended file size limits.

Hosting the datasets on Kaggle provides:

- Faster downloads
- Version management
- Reliable storage
- Easy access for researchers and developers

---

# 🚀 Using the Dataset

After placing the datasets in the `data/` directory, you can:

1. Train the Traditional Deep Learning Models.
2. Train the Advanced Deep Learning Models.
3. Run the FastAPI web application.
4. Reproduce all experiments and evaluation results.
